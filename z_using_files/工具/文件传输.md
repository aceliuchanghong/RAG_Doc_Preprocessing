### http 服务

下载服务

```bash
python -m http.server 8080
```

---

`uploadserver` 在 Python 自带服务的基础上增加了上传页面，且支持浏览器批量选择图片和视频。

1.  **在电脑端安装：**
    打开终端（CMD 或 Terminal），输入：
    ```bash
    pip install uploadserver
    ```

2.  **启动服务：**
    ```bash
    python -m uploadserver
    ```
    *此时终端会显示 `Serving HTTP on 0.0.0.0 port 8000`。*

3.  **访问：**
    浏览器访问：`http://电脑IP:8000/upload`。

4. **进阶**:
    ```bash
    python -m uploadserver --port 9000
    python -m uploadserver 9000 --token mysecret
    ```
